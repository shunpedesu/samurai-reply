## ビジネスコンテキスト

### なぜこのアプリか
- RIZZ AI（恋愛返信特化）が月$500K稼いでいる
- 「スクショを友人に送って返信を相談する」という行動はすでに存在 → 需要証明済み
- 日本語市場では「笑える＋使える」の組み合わせはまだ誰も取っていない
- Rage版のスクショがSNSに投稿されることで勝手に拡散する設計

### 差別化ポイント
- Rage（DO NOT SEND）版 → スクショSNS投稿でバズる
- 侍バージョン → 海外でのユニーク性・話題性
- スクショOCR → LINEの画面をそのまま読み込める
- $9.99買い切り → 「サブスクなし」がSNSで拡散しやすい

## 技術的な注意点

- Stripe Webhook は raw body が必要 → `express.raw()` を `/api/stripe/webhook` に適用済み
- `express.json()` は Webhook エンドポイント以外に適用
- Supabase は service_role キーを使用（RLS をバイパスするため）
- フリー制限は IP + 日付のキーで Supabase に保存（将来 Redis に移行可）
- Claude API キーはバックエンドのみに置く（フロントエンドには露出しない）
